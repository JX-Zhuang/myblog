# 基于Express、Mongoose的多人博客
## 一、博客初始化
### npm install express-generator -g
全局安装Express应用生成器
### express -e myblog
初始化ejs模版的express
### npm install
安装依赖
## 二、创建git仓库
* git init
* git add .
* git commit -m 'add'
* git remote add origin https://github.com/JX-Zhuang/myblog.git
* git push -u origin master
